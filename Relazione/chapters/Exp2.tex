\chapter*{Esperimento 2: Timer e LED con interrupt}
\addcontentsline{toc}{chapter}{Esperimento 2: Timer e LED con interrupt}

\section*{Obiettivo}
Far lampeggiare un led ad una frequenza stabilita e accendere/spegnere un altro led in base alla pressione di un bottone utilizzando gli interrupt.

\section*{Svolgimento\footnote{Nella repository è il progetto Exp02}}
Tramite CubeMX impostiamo il TIM6 con i valori calcolati nell'esperimento precedente per avere una frequenza di 1Hz.
Dopo di che abilitiamo l'interrupt del TIM6.

\begin{minted}
[
frame=lines,
framesep=2mm,
baselinestretch=1.2,
fontsize=\footnotesize,
]{C}
//enable counter
TIM6->CR1 |= TIM_CR1_CEN;
//enable interrupt
TIM6->DIER |= TIM_DIER_UIE;
\end{minted}

Nella routine di interrupt di TIM6 andiamo ad accendere e spegnere il LED1 utilizzando la periferica GPIO, come visto nell'esperimento precedente.

\begin{minted}
[
frame=lines,
framesep=2mm,
baselinestretch=1.2,
fontsize=\footnotesize,
]{C}
void TIM6_DAC_IRQHandler(void)
{
  /* USER CODE BEGIN TIM6_DAC_IRQn 0 */
	if(GPIOB->ODR & GPIO_ODR_OD0_Msk){
				//turn off led1
				GPIOB->ODR = GPIOB->ODR & ~GPIO_ODR_OD0_Msk;
	} else {
				//turn on led1
				GPIOB->ODR = GPIOB->ODR | GPIO_ODR_OD0_Msk;
	}
  /* USER CODE END TIM6_DAC_IRQn 0 */
  HAL_TIM_IRQHandler(&htim6);
  /* USER CODE BEGIN TIM6_DAC_IRQn 1 */

  /* USER CODE END TIM6_DAC_IRQn 1 */
}
\end{minted}

Usiamo poi la periferica EXTI per gestire la pressione del bottone.
Il bottone è collegato al pin PC13: selezioniamo la modalità EXTI13 dopo aver cliccato sul pin, e nel menù GPIO abilitiamo l'interrupt associato.
Routine di interrupt EXTI andiamo ad accendere e spegnere il secondo led.
\begin{minted}
[
frame=lines,
framesep=2mm,
baselinestretch=1.2,
fontsize=\footnotesize,
]{C}
void EXTI15_10_IRQHandler(void)
{
  /* USER CODE BEGIN EXTI15_10_IRQn 0 */
 	if(GPIOB->ODR & GPIO_ODR_OD7_Msk){
				//turn off led2
				GPIOB->ODR = GPIOB->ODR & ~GPIO_ODR_OD7_Msk;
	} else {
				//turn on led2
				GPIOB->ODR = GPIOB->ODR | GPIO_ODR_OD7_Msk;
	}
  /* USER CODE END EXTI15_10_IRQn 0 */
  HAL_GPIO_EXTI_IRQHandler(GPIO_PIN_13);
  /* USER CODE BEGIN EXTI15_10_IRQn 1 */

  /* USER CODE END EXTI15_10_IRQn 1 */
}
\end{minted}
